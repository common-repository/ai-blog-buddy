=== Introduction ===

Introducing AI Blog Buddy, the OpenAI GPT-3 Plugin for WordPress! Now you can use the power of OpenAI to create high-quality content in a matter of seconds. This plugin allows you to write posts and pages with the help of OpenAI GPT-3 models, making content creation easier and faster. With this plugin, you can quickly create engaging and interesting content without the hassle of manual writing. Get the OpenAI GPT-3 Plugin now and start creating amazing content for your WordPress website!
=== Features ===

- Meta box with input fields for model, API key, prompt, temperature, max tokens, top p, frequency penalty, and presence penalty
- Send button to send API requests to the OpenAI server
- Ability to append the response to the current post or page content

=== Requirements ===

- WordPress 4.0 or higher
- OpenAI API key

=== Installation ===

1. Download the plugin from the WordPress plugin repository.
2. Log in to your WordPress site as an administrator.
3. Navigate to the "Plugins" section of your WordPress dashboard.
4. Click the "Add New" button.
5. Click the "Upload Plugin" button.
6. Select the plugin zip file you just downloaded and click the "Install Now" button.
7. After the plugin is installed, click the "Activate" button.

=== Usage ===

1. Log in to your WordPress site as an administrator.
2. Create a new post or page, or edit an existing one.
3. Scroll down to the "OpenAPI Chat" meta box.
4. Enter your OpenAI API key in the API key field.
5. Enter a prompt in the prompt field to provide context for the GPT-3 model.
6. Adjust the temperature, max tokens, top p, frequency penalty, and presence penalty fields as desired.
7. Click the "Send" button to send the API request to OpenAI.
8. The response from OpenAI will be appended to the current post or page content.
